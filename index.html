Javascript Components

In order to install javascript components, you can use Bower and then add this line:

<link href="auth0-styleguide/index.css" rel="stylesheet" />
<script src="auth0-styleguide/index.js"> </script>
Tooltip
As HTML:
<div data-ot="The content" data-ot-delay="2" data-ot-hide-trigger="closeButton">Hover me</div>
As Javascript:
var Opentip = require('styleguide').Opentip;
new Opentip("#my-trigger-element", "Optional content", "Optional title", { ...options... });
